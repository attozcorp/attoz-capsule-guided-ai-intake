# Micro Vertical Intake

Micro Vertical Intake is the public proprietary method used by Attoz Capsule to adapt guided AI intake to specific local service niches.

Attoz Capsule is a guided AI intake system for local service businesses, powered by Micro Vertical Intake.

## Public definition

Micro Vertical Intake means that a service request should be understood through the context of the niche. A landscaping customer, plumbing customer, roofing customer, HVAC customer, and mobile mechanic customer are not asking the same kind of question.

Each niche has different signals. Some requests are urgent because water is active. Some require access to a yard. Some need vehicle location and symptoms. Some need optional photos to clarify visible conditions.

## Why niche context matters

A generic form can collect name, phone, email, and message. That is often not enough. The owner still has to interpret the job, ask clarifying questions, and decide whether the request needs immediate attention.

Micro Vertical Intake improves the public intake frame by making the request path fit the service category.

## What it does publicly

Micro Vertical Intake helps explain why the intake flow should change by service niche.

For example:

1. A landscaping request may need property type, service area, yard condition, timing, and optional photos.
2. A plumbing request may need leak status, affected fixture, water shutoff context, and urgency.
3. A mobile mechanic request may need vehicle year, make, model, symptoms, location, and whether the vehicle starts.

These examples are illustrative. They do not describe private implementation logic.
