# Contributing

This repository is maintained as public concept documentation for Attoz Capsule.

Do not submit product source code, prompts, internal workflows, scoring systems, routing logic, database structures, customer data, screenshots from private systems, cloning instructions, credentials, API keys, tokens, logs, or environment variables.

Attoz Corp may reject contributions that expose intellectual property, create security risk, confuse the product category, imply open source availability, or describe how to recreate private product behavior.

All contributions are subject to review.
